This project demonstrates how to host multiple independent websites on a single Amazon EC2 instance using Nginx virtual hosts. Each website is linked to its own domain via Route 53 (or custom DNS) and served from a separate directory under /var/www/.

üõ†Ô∏è Setup Steps

  1. Launched an Amazon Linux EC2 with ports 22 & 80 open.
  2. Installed Nginx via amazon-linux-extras enable nginx1.
  3. Created separate folders under /var/www/ for each site.
  4. Placed unique HTML files in each directory.
  5. Added individual .conf files in /etc/nginx/conf.d/.
  6. Restarted Nginx to apply the virtual-host configs.
  7. Mapped test domains in local /etc/hosts or Route 53.
  8. Verified that each domain served its own content.
